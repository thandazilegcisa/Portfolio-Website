Project S'ghubu

Html Css Javascript

Project S'gubhu is a music participation web application that turns a users mobile phone motion gestures into beats. This is an experimental project that explores interaction design. For the sake of providing alternative control stystems for musical instruments through gesture control.

The project uses various api's, namely the fetch api, web audio api, and most importantly the accelerometer api. As such you will need a mobile phone in order to test the project out for yourself.
